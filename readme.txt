== Description ==

云采编(wp-cloud-edit) 是简单却高效的文章发布工具。你可以方便的发布文章，尤其是当下网络上热门文章。
Simple but effective plugin allows you to post articles. You can easily publish articles, especially the present popular online articles.

Major new features in wp-cloud-edit 1.0 include:
*hot articles
*one-click publish
*你可以看到当下互联网上热门的文章
*而且文章已经自动分好类，你可以通过类别查询
*你可以创建自己的规则，通过关键词进行筛选
*你可以将规则跟网站现有类别进行关联，进行一键发布
*更多详情请移步 云采编 官方网站(http://shuzitansuo.com/ycb.html)

PS:You'll need an [API token](http://shuzitansuo.com/contact.html) to use it.


== Installation ==

Upload the wp-cloud-edit plugin to your blog, Activate it, then enter your [solr.com.cn API token](http://solr.com.cn/get/).

1, 2, 3: You're done!
